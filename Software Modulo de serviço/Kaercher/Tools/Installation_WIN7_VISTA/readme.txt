ENGLISH:

rem ++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++
rem + Notes on how to activate "ACTIVE-X Components for Kaercher Serviceprogram" with WINDOWS VISTA AND 	 +
rem + WINDOWS 7 operating systems                                                                  		 +
rem +                                                                                                            +
rem + 0. Make sure you have been granted maximum administrator rights for your computer before continuing	 +
rem + 1. Copy Batch-file "regocx" to directory c:\windows\system32 	                                 	 +
rem + 2. Start COMMAND WINDOW in administrator mode! To do this select one of the following methods:             +
rem +														 +
rem +    Method 1: Click on Start-button of Windows task bar -> Next type "cmd" in input field -> Press "CTRL    +
rem +              SHFIT + ENTER"-Keys simultaneously. You enter "COMMAND WINDOW" as administrator.              +
rem +    Method 2: Use Windows explorer to change to directory "c:\windows\system32" and click on file "cmd.exe" +
rem +              with RIGHT MOUSE KEY. A system box asks how to execute this file. Click on "Execute as        +
rem +              Administrator"										 +
rem +    Method 3: Like Method 2, but create a shortcut icon on desktop.				         +
rem +              Execute this shortcut by clicking on icon with RIGHT MOUSE KEY and select "Properties ->      +    
rem +              -> Advanced -> Shortcut -> Execute as Administrator"                                          +
rem +														 +
rem + 3. In COMMAND WINDOW type "regocx"		                                                 	 +
rem + 4. Confirm each successful OCX-registration by clicking on "OK"-button of window.				 +
rem + 5. Step 4. has to be repeated for 6 files.							         +
rem + 6. Now you're all set.											 +
rem ++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++
